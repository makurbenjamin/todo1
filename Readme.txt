github.com - create repo


git init		- Starts git and creates .git
touch Readme.txt	- creates a file
git status		- gives status
git add Readme.txt	- adds Readme.txt for commit
git status
git commit -m "Add..."	- Commit
git remote add origin https://github.com/makurbenjamin/todo1.git - Connects to github
git remote -v		- shows current repo
git push -u origin master	- uploads commit
